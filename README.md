## Registration form with PHP backend

A modern, responsive registration form with PHP backend functionality that allows users to register, view registered users, and manage user data with edit/delete operations.

## Features

- **User Registration**: Secure user registration with form validation
- **User Management**: View all registered users in a clean interface
- **Edit & Delete**: Update user information or remove users
- **Responsive Design**: Works on desktop and mobile devices
- **PHP Backend**: Robust server-side processing with MySQL database
- **Password Security**: Passwords are securely hashed using PHP's `password_hash()`

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript, Tailwind CSS
- **Backend**: PHP
- **Database**: MySQL
- **Styling**: Custom CSS with Google Fonts (Poppins)
